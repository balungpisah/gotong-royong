Token Reference
CSS variable names for implementation. All tokens defined in :root.
Type Scale: --fs-display 32px · --fs-h1 26px · --fs-h2 20px · --fs-h3 16px · --fs-body 14px · --fs-small 12px · --fs-caption 11px · --fs-micro 9px

Line Height: --lh-tight 1.2 (headings) · --lh-normal 1.5 (body) · --lh-relaxed 1.7 (long text)

Font Weight: --fw-regular 400 · --fw-semi 600 · --fw-bold 700 · --fw-extra 800

Spacing: --sp-1 4px · --sp-2 8px · --sp-3 12px · --sp-4 16px · --sp-5 20px · --sp-6 24px · --sp-8 32px · --sp-10 40px · --sp-12 48px · --sp-16 64px

Radii: --r-sm 6px · --r-md 10px · --r-lg 14px · --r-xl 20px · --r-full 9999px

Shadows: --shadow-sm badges/pills · --shadow-md cards · --shadow-lg popovers · --shadow-xl modals

Usage Guide:
Card padding: --sp-4 (16px) inner body, --sp-3 (12px) meta/footer
Card gap in feed: --sp-3 (12px) between cards
Section gap: --sp-8 (32px) between sections
Card title: --fs-h3 + --fw-bold + --lh-tight
Card body: --fs-body + --fw-regular + --lh-normal
Stepper text: --fs-micro + --fw-semi
Button text: --fs-small + --fw-semi
Track header: --fs-caption + --fw-extra + uppercase + 1.2px letter-spacing
