## 3. Stages Per Track

### 3.1 Tuntaskan (Fix It)

| Stage | Name | Purpose | Card Face |
|---|---|---|---|
| Seed | Keresahan | Initial signal: "I see a problem" | Testimony post + co-witness dots |
| Discuss | Bahas | Define the problem, diagnostic checklist, assign PIC | Discussion thread + criteria draft |
| Plan | Rancang | Design the plan, break into tasks (optional for simple fixes) | Task plan + approval votes |
| Execute | Garap | Do the work: tasks, heartbeats, evidence | Task board: done/active/open |
| Verify | Periksa | Verify the fix works, challenge window | Verification panel + timer |
| Done | Tuntas | Archive, final state | Completed banner + CV rewards |

Branching at Bahas: Simple fix (skip Rancang), Complex project (full path), Funded (add Galang), Advocacy (add Siarkan).

### 3.2 Wujudkan (Build It)

| Stage | Name | Purpose | Card Face |
|---|---|---|---|
| Seed | Gagasan | "Let's build this" | Idea post + supporter count |
| Discuss | Bahas | Refine the vision, feasibility, assign PIC | Discussion thread |
| Plan | Rancang | Design it, tasks, estimate resources | Blueprint + task breakdown |
| Fund | Galang | Pool resources (optional but common) | Funding tracker |
| Execute | Garap | Build it: tasks, heartbeats, evidence | Task board |
| Celebrate | Rayakan | Celebrate the creation | Celebration + showcase |
| Done | Tuntas | Archive | Completed banner + CV rewards |

### 3.3 Telusuri (Explore It)

| Stage | Name | Purpose | Card Face |
|---|---|---|---|
| Seed | Pertanyaan | "Why does this happen?" | Question post |
| Hypothesize | Dugaan | Propose candidate explanations | Hypothesis list + discussion |
| Test | Uji | Research, experiment, observe | Experiment tracker + evidence |
| Findings | Temuan | Best current answer + confidence level | Findings summary |
| Done | Tuntas | Community knowledge or track change | Knowledge card |

Exit paths from Temuan: Problem found → Tuntaskan, Idea emerged → Wujudkan, Knowledge gained → stays Tuntas.

### 3.4 Rayakan (Honor It)

| Stage | Name | Purpose | Card Face |
|---|---|---|---|
| Seed | Kabar Baik | "Something great happened" | Celebration post |
| Affirm | Sahkan | Community validates the achievement | Affirmation count + endorsements |
| Recognize | Apresiasi | Public recognition, badges | Recognition showcase |
| Impact | Dampak | Track lasting positive impact (optional) | Impact attestations |
| Done | Tuntas | Archive | Legacy card |

### 3.5 Musyawarah (Decide Together)

| Stage | Name | Purpose | Card Face |
|---|---|---|---|
| Seed | Usul | "I propose we decide on X" | Proposal card |
| Discuss | Bahas | Deliberate, present options, debate | Discussion thread + option list |
| Decide | Putuskan | Community votes on the decision | Voting panel + results |
| Enact | Jalankan | Implement the decision, assign responsibilities | Assignment list + timeline |
| Review | Tinjau | Review effectiveness (optional) | Review dashboard |
| Done | Tuntas | Decision archived as community record | Decision record |

Decision spawns action: A Musyawarah decision can auto-create linked entities in Tuntaskan or Wujudkan.
