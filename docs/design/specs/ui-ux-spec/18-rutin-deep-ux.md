## 17. Rutin Deep UX

### 17.1 Spawn Points Per Track

| Track | Spawn Stage | Rationale |
|---|---|---|
| Tuntaskan | Garap | Plan exists from first cycle |
| Wujudkan | Garap | Execution stage |
| Telusuri | Uji | Recurring investigation |
| Rayakan | N/A | Celebrations are unique |
| Musyawarah | Jalankan | Decision made, implementation repeats |

### 17.2 Inheritance & Reset

Inherits: PIC (or rotated), task template, schedule, membership, Rahasia level. Resets: votes, evidence, progress, timestamps. Links: bi-directional to previous cycle.

### 17.3 PIC & Task Rotation

Fixed, round-robin, volunteer. Fairness tracking. Skip-cycle: propose → consent 24h. 3 consecutive skips → flag relevance.
