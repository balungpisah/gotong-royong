## 5. Cross-Cutting Capabilities

### 5.1 Galang (Pool Resources)

Mini-lifecycle: Sasaran → Kumpul → Salurkan → Lapor. Bendahara manages. Card shows: target, collected, contributors, transparency log.

### 5.2 Siarkan (Broadcast / Amplify)

Reach tracker, share actions, media kit, Humas role.

### 5.3 Rutin (Recurring)

Frequency: harian/mingguan/bulanan/custom. Set at Bahas or Rancang. Auto-spawn 48h before at track-specific execution stage. PIC rotation: fixed, round-robin, volunteer.
