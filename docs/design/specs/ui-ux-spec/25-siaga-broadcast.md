## 24. Siaga Broadcast (NEW)

Emergency alert system — speed over ceremony.

### 24.1 Entry

Via AI-00 siaga-ready context bar state.

### 24.2 Siaga Palette

Uses siaga colors: `--s-deep` (#B71C1C) through `--s-border` (#FFCDD2). Red pulse animations.

### 24.3 Four States

| State | UI | Real-time Elements |
|---|---|---|
| Kirim | Compose: pre-filled text, emergency type chips, auto-location card, 112 link | Siarkan Sekarang button |
| Aktif | Live card with shimmer, real-time stats, quick update input, event timeline | terjangkau/melihat/merespons counters |
| Respons | Responder cards (distance/ETA/status), quick-respond button, timeline | Real-time responder updates |
| Selesai | Deliberate confirmation, summary (duration/responders/services), thank you | Green resolved bar |

### 24.4 Broadcast Bar

Bottom bar morphing through states: Composing → Active (pulsing) → Resolved (green). 112 emergency link always visible across all states.

### 24.5 Emergency Type Chips

Quick-select: Kebakaran, Banjir, Gempa, Darurat Medis, Kecelakaan, Keamanan, Lainnya.

### 24.6 Reputation

Zero Tandang credit for Siaga broadcasts. Emergency response is civic duty, not reputation-building.
