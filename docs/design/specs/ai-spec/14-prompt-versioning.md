### 14.2 Prompt Updates & A/B Testing

- Prompts updated on 2-week cadence (review every other Friday)
- Changes require approval from AI Lead + Community Lead
- New prompt versions tested on 5–10% traffic before rollout
- Previous prompt versions retained for audit trail
